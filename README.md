# Campus Booking System

A web application that allows students, faculty, and administrators to efficiently manage and book campus resources such as seminar halls, labs, and classrooms. This project provides a user-friendly interface, real-time booking updates, and an admin dashboard for smooth management.

---

## **Features**

### Student
- Register and login using college credentials (USN/email).  
- View available resources (labs, seminar halls, classrooms).  
- Book slots for resources and check booking history.  
- Receive confirmation notifications for bookings.  

### Faculty
- View all resource bookings.  
- Approve or reject student booking requests (optional).  
- Direct messaging with students for resource-related queries.  

### Admin
- Access full dashboard with all bookings.  
- Analyze usage statistics.  
- Manage users, resources, and announcements.  
- Monitor booking conflicts and resource availability.  

---

## **Tech Stack**

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Node.js, Express.js  
- **Database:** MySQL  
- **Hosting/Version Control:** GitHub
